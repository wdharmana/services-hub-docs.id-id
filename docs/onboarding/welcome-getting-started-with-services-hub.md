# <a name="getting-started-with-the-services-hub"></a>Memulai Services Hub

## <a name="onboarding-your-team-to-the-services-hub"></a>Mengarahkan Tim Anda ke Services Hub

Microsoft Service Hub adalah portal dukungan yang memberikan visibilitas ke semua produk dan layanan Microsoft Anda, pelatihan dan sumber daya dukungan yang disesuaikan, dan pemantauan solusi untuk membantu mencegah dan menyelesaikan masalah secara efisien.

Untuk mempercepat, pastikan untuk membahas dengan tim Anda mengenai navigasi inti, dasbor, dan di mana mereka dapat menemukan informasi lebih lanjut tentang semua fitur Services Hub.. Semua informasi tersebut ada di dalam panduan ini.

### <a name="to-get-started-follow-these-four-steps"></a>**Untuk memulai, ikuti empat langkah berikut:**

1.  Daftar Services Hub dan tinjau [email pendaftaran Services Hub](https://microsoft.sharepoint.com/teams/ASD_Communications/CR/_layouts/OneNote.aspx?id=%2Fteams%2FASD_Communications%2FCR%2FDocuments%2FASD%20Release%20Guide%20Master&wd=target%28Services%20Hub%20Onboarding.one%7C424CD822-86D3-432D-AA1C-A2B158FDA370%2FRegistration%20email%7CEE440970-7D59-4CFD-AAAF-5A9D09623CDC%2F%29).

2.  [Undang orang lain di tim Anda.](https://microsoft.sharepoint.com/teams/ASD_Communications/CR/_layouts/OneNote.aspx?id=%2Fteams%2FASD_Communications%2FCR%2FDocuments%2FASD%20Release%20Guide%20Master&wd=target%28Services%20Hub%20Contract.one%7C02D7E5B3-F7E0-49E5-925A-54F5B21900E8%2FManage%20Access%7CC6BBC243-950E-474B-92CF-1364D9E3BDCC%2F%29) Pastikan untuk mengundang orang lain di tim Anda ke Services Hub yang akan menjadi kontak dukungan yang Anda tetapkan. Kemudian atur izin di halaman Kelola Akses untuk mengizinkan mereka melihat ID akses dan kata sandi mereka. Ikuti petunjuk untuk memulai ID akses di [Microsoft Premier Online](https://go.microsoft.com/fwlink/?linkid=838933).

3.  Saksikan [video gambaran Umum Services Hub](https://serviceshub.microsoft.com/about/) lalu bagikan dengan tim Anda.

4.  Identifikasi administrator Azure Anda dan [mulailah penilaian atas permintaan.](https://microsoft.sharepoint.com/teams/ASD_Communications/CR/_layouts/OneNote.aspx?id=%2Fteams%2FASD_Communications%2FCR%2FDocuments%2FASD%20Release%20Guide%20Master&wd=target%28Services%20Hub%20Health.one%7CF0CB74B8-AEC6-47A6-9F82-E15A0E3B33D3%2FGetting%20started%20with%20on-demand%20assessments%7CFC422F63-0BCC-4A82-B9DF-B77F73FBA35C%2F%29).
