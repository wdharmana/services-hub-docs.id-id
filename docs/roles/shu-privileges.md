# <a name="services-hub-users-unique-privileges"></a>Keistimewaan Unik Pengguna Services Hub

## <a name="services-hub-users-have-unlimited-access-to-the-services-hubs-online-digital-experiences"></a>Pengguna Services Hub memiliki akses pengalaman digital online Services Hub tanpa batas.

> Mengundang pengguna sebanyak-banyaknya untuk bergabung dengan Services Hub

> Melihat katalog layanan online dan meminta informasi tentang suatu layanan

> Melihat deskiripsi kontrak

> Melihat informasi kontak Microsoft

> Mengikuti produk di pusat pembaruan

> Mengikuti kelas menggunakan pembelajaran atas permintaan

> Lihat dan berpartisipasi dalam webcast langsung atau rekaman

> Membuat rencana untuk sendiri atau orang lain.
